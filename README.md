# binary-bot
Visual automation for binary.com [bot.binary.com](https://bot.binary.com)

Visit [wiki](https://github.com/binary-com/binary-bot/wiki) for more info.

## Development

```
npm install
npm run dev
```

## Deployment 

```
gulp test-deploy
npm run deploy
```

## To update to latest version

```
git pull
npm update
```
